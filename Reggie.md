Feet (make 4)
Black yarn
Magic ring
1. 6sc into MR (6)
2. Inc into each sc (12)
3. (Sc, inc) x 3 (18)
4. Sc all round (18)
5-7) Repeat row 4
Change to brown 
8) sc all around 
9-15) repeat row 8
Note: if you want longer legs, repeat row 8 until desired length is reached.
Bind off leaving long tail for stitching.
Stuff legs and close. Leave tail to stitch onto body.
